# Skill: Frontend (framework-agnostic)
- Keep components pure; derive state; avoid prop drilling (use context/store when needed).
- Accessibility: labels, roles, focus order, keyboard nav.
- Data fetching: cache by full parameter set; handle loading/error/empty states.
- Perf: avoid unnecessary re-renders; memoize heavy ops; code-split routes/pages.
- Tests: critical flows via component + e2e tests.
